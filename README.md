# Home

## Run Locally

`bundle exec jekyll serve`

## New Post

Format is https://jekyllrb.com/docs/posts/#creating-posts

YEAR-MONTH-DAY-title.MARKUP

The post will not show in the index until the date of the post!

## Publishing

Commit and push to `main` branch and GitHub will update the website automatically

## Access

You've created this under your nik@osvalds.net GitHub account so you many need to grant access to `@nosvalds` if you can't push.
